== What is Gradle?

=== Gradle's purpose

Gradle is a build and automation tool.

[%step]
* JVM based
* Implemented in Java
* 100% Free Open Source - Apache Standard License 2.0

=== Agnostic Build System

* Java ecosystem
** Groovy, Kotlin, Scala, ...
* Android
* Native ecosystem
** C, C++, Swift, ...
* Misc
** Go, Asciidoctor, ...
